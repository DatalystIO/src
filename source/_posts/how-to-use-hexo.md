## Environment Setup

### Step 1: Install Nodejs via `nvm`

```
# wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
```

```
source .bashrc
```

```
nvm install stable
```


### Step 2: Install Hexo


```
npm install -g hexo-cli
```

### Step 3: Install Hexo plugins

hexo-deploy-git

```
npm install hexo-deployer-git --save
```
